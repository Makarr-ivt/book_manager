# Book Manager
Консольное приложение для управления библиотекой книг на C++. Создано в рамках курсовой работы по ООП, ВГТУ 2025.

## Функционал
- Добавление/удаление/редактирование книг
- Поиск по автору, году, количеству страниц
- Валидация ввода данных

## Сборка
```shell
mkdir build
cd build
cmake ..
cmake --build .
```
Или используйте функции своей среды разработки для запуска программы. В зависимости от системы могут потребоваться разные дополнительные действия, такие как настройка кодировки терминала и т.п. Имейте это в виду при установке!

## Использование
Запустите программу и следуйте инструкциям меню.
